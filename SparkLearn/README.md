###### 常用大数据框架：
    spark,hadoop,flink,hive

###### hadoop包括3部分: 
    1)分布式并行计算框架mapreduce,
    2)分布式文件系统hdfs, （目前看,要会基本的使用命令）
    3)分布式资源调度系统yarn

###### spark包括：
    1）spark core: 基本离线处理技术 （目前看）
    2）spark streaming: 实时处理技术
    3）spark sql: sql形式处理数据
    4）spark mllib: 机器学习库
    5）spark graphx: spark提供的图算法库
